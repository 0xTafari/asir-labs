# ASIR Labs | System and Network Administration

Este repositorio contiene prácticas realizadas durante mi formación en **Administración de Sistemas Informáticos en Red (ASIR)**, incluyendo la instalación, configuración y securización de sistemas y servicios de red.

El objetivo de este repositorio es documentar los conocimientos adquiridos y servir como referencia para futuras implementaciones en entornos reales y de ciberseguridad.
